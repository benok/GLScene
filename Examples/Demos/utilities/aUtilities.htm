Utilities :

  • joystick :
    • using the TJoystick component to get joystick position and button states

  • keymap :
    • using the keyboard unit to check key states asynchronously
    • using mapped virtual key codes and key names

  • HTFUtils :
    • two utilities to generate and view HTF files (TerrainPack and HTFViewer)
    • using HTF to compress elevation data and later extract/access that data

  • GuiEditor :
    • An editor for combining several GUI-Component into one layout

  • recorder :
    • using the TAviRecorder component to record a GLScene animation with an AVI codec

  • screensaver :
    • using the TScreenSaver component for making a screen-saver
    • full featured screen-saver, preview, GLScene-based properties interface and password support
    • using TGLCadencer and behaviours for animation
