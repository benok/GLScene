Benchmarks :

  • canvas :
    • drawing 2D with TGLCanvas in a scene viewer
    • allows benchmarking TGLCanvas performance vs GDI
    • hardware-acceleration on both sides is available

  • megacube :
    • a high number of shaded cubes arranged in a cubic array
    • allows benchmarking raw geometrical and T&L power (no texturing here)

 

  • megaglasscube :
    • a high number of shaded transparent cubes arranged in a cubic array
    • same use as megacube but with emphasis on transparency and depth sorting

  • volcano :
    • volcano-like particle systems using ParticleFX
    • benchmarking of PFX and depth sorting
    • PFX is a scene-wide blended particles system

  • smoking :
    • several heavily smoking fire PFXs
    • benchmarking of PFX sprites, depth sorting
    • intertwined additive/transparently blended PFXs

  • whirlwind :
    • a whirlwind-shaped particle systems (via TGLParticles)
    • allows benchmarking particle systems
    • object management (not even shading is used here)
